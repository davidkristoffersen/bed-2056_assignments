# bed-2056_assignments

## Requirements

**Cli packages**:
- python3
- pip3

**Pip packages**:
- matplotlib
- statistics
- urllib3
- pandas
- beautifulsoup4

## How to run

### Assignment 1
```bash
$ cd assignment-1_global-temperature
$ python3 main.py
```
